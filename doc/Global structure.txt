This project is meant to be used for a futur chat projet.


EmosAnalyser is divided in 2 parts:

- Analyser (in Java)
- A custom dictionary 


The analyser will contain:

- A point system
- A word-by-word checking system
- A enum of possible emotion

The words in the dictonary:

- Type (adverbs, adjectives, nouns...)
- Points attributions
- Special kinds of words
